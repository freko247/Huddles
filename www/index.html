Introduction

This application is used to connect with people who have common interests, goals or purposes in life.

This application is heavily profile dependant, so please create a profile to utilize the application.

No information about you or your profile will be shared with 3rd parties.

The application utilizes information you input to best fit the use of the application to you.

Navigation

The application panel is always accessible and is the core navigation feature of the application.

Sometimes tabs will appear which will reveal more information within a certain page. Clicking these will change the interface to display the proper information.

Skills

To help you find people to collaborate with, the application utilizes a set of skills. Skills are editable inside the profile and will help you and other people to identify if you/they are a group match.
